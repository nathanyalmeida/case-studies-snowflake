# ❄️ Case Studies – Snowflake

This repository contains hands-on case studies, exercises, and reference materials developed while following different courses and learning paths.

The goal is to strengthen practical knowledge of Snowflake by working through real-world scenarios involving data ingestion, transformation, and analysis.

## 🎯 Objectives

- Practice **Snowflake fundamentals**
- Work with **real-world datasets**
- Improve **SQL and data modeling skills**
- Explore **data pipelines and transformations**
- Build a portfolio of **case studies**

## 🧠 Topics Covered

- Snowflake architecture (databases, schemas, warehouses)
- Data loading and staging
- SQL transformations
- Analytical queries
- Performance considerations
- Basic data engineering workflows

## ⚙️ Technologies Used

- Snowflake
- SQL
- Git & GitHub
- (Optional) Python / Snowpark
